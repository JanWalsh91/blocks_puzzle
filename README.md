# Blocks Puzzle

### 2D Unity game: Move blocks across a puzzle.
*Ecole 42 Piscine Unity Project*

##### Instructions: 

- Use 1, 2 and 3 to switch between characters. 
- Move around with arrow keys. Use space to jump.
- Move all the blocks to their end boxes!
